Process and result:

Usability Testing also known as User Experience (UX) Testing, is a testing method for measuring how easy and user-friendly a software application is. A small set of target end-users, use software application to expose usability defects. Usability testing mainly focuses on user’s ease of using application, flexibility of application to handle controls and ability of application to meet its objectives.
As I did the user testing before for the project website I tried to follow those instruction for this assignment as well. For testing the Figma prototype, the method that I chose for this test was remote moderated tests, that means I did not have this ability to visit my users in person and we were physically in different places and I sent them the link of my prototypes to my users. Also, I did the testing in person too both for first test and the second one.
