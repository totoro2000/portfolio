Task analysis

Define phase

For the task analysis, I chose Duolingo as an example. The main task for using this website or its app is learning and improving language skills and then according to the different features that Duolingo has I create subtasks:

- Create a premium account.
- Find other people who are learning the language you are learning.
- Add another language.
- Get more information about the language that you are going to learn.

Questions:

1. What is task analysis?
2. What is hierarchical task analysis?
3. Why we should use task analysis?
The CMD methos that I used for this assignment are:
- Customer journey
- Literature study

Learning outcomes:

• Communication:
For this assignment at first I had problems with separating each task from anther one, what should I mention as a task, how to create subtask for each task, after getting feedback from my mentor the process become more obvious for me.
• Research :
For understanding this subject better I searched for more examples and information about task analysis.
