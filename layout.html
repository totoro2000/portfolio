Magazine layout

This week we learn about the layout and an introduction to InDesign for the assignment of this session we had a group project to choose a subject for a magazine and design a layout for it. We chose a fashion magazine and the feedback that we got from the teacher was good, the only thing that we should pay more attention to it is not using some photos like portraits between two pages.

Learning outcomes:

Communication:
Work as a group to design a magazine layout in a limited time.
Design:
Learn more about Adobe Indesign, and page layouts.
